# Hodoplex
Hodophlex is a hybrid web app for Amazon Flex drivers to track earnings, mileage, expenses, blocks, and generate custom reports for taxes and planning.

## Features
- Track earnings, mileage, expenses, incidents, and more
- Log multiple blocks per day and link them to vehichles and stations
- Generate and download reports(PDF, print, doc)
- Visual performance graphs and summaries
- Manual entries and error recovery
